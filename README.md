# Gojo vs Sukuna – Data Analysis

A data-driven breakdown of the battle between the "Strongest of Today" and the "Strongest in History" using Python.

## Project Overview

This project analyzes key aspects of the fight using structured data and custom metrics.  
The goal was to explore how raw power, energy efficiency, and adaptation impacted the outcome.

## Tools Used

- Python
- Pandas
- Matplotlib

## Key Insights

- Raw damage output was nearly identical.
- Gojo demonstrated higher energy efficiency.
- Adaptation-based techniques created the decisive advantage.

## Technical Focus

- Data grouping & aggregation
- Custom metric creation (efficiency score)
- Comparative analysis
- Data storytelling through visualization

## Conclusion

The strongest didn’t lose because he was weaker —  
he lost because his opponent evolved.
